Element Properties

1
Position & Size

X Y Height are read-only — computed by Fabric. Width is editable. Synced whenever selection changes.

2
Number — auto + override

Scans all numbered paragraphs sorted by Y, computes number automatically. Shows auto badge. User can type a manual value — sets data.numberingOverride. "↺ reset" clears it.

3
Paragraph Text — two-way

Textarea shows raw text with markdown markers (**bold**). Panel edits call obj.set('text', …). Canvas text:changed event updates the textarea live.

4
Indent Level dropdown

Options: 0 — Main, 1 — Sub-item (2.1), 2 — Sub-sub (2.1.1), 3 — Nested (2.1.1.1). Maps to data.level and applies existing indent/tabStop presets.

5
Top bar — trimmed for paragraphs

Keeps only Bold Italic Align Color. Font size and numbering move to this panel. Floating ParagraphToolbar removed.
