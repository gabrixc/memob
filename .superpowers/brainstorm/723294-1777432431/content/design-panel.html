Sifat Elemen

1
Position & Size

X Y Tinggi are read-only — computed by Fabric. Lebar is editable (sets textbox width on canvas). Synced whenever selection changes.

2
Nombor — auto + override

Default: scans all numbered paragraphs on the page sorted by Y, computes the number automatically. Shows auto badge. When user types a value, it sets data.numberingOverride and shows override badge. "↺ reset" clears the override.

3
Teks Perenggan — two-way

Textarea shows raw markdown-style text (e.g. **bold**). Editing in the panel calls obj.set('text', …) on the Fabric object. Canvas text:changed event updates the panel textarea in real-time when user types directly on canvas.

4
Aras Inden dropdown

Values: 0 — Utama, 1 — Sub-item (2.1), 2 — Sub-sub (2.1.1), 3 — Dalam (2.1.1.1). Maps directly to data.level (0–3). Applies the same indent/tabStop presets as existing ParagraphToolbar level buttons.

5
Top PropertiesBar — trimmed

For paragraph elements, the top bar keeps only: Bold Italic Align Color. Font size and numbering controls move exclusively to the right panel. Floating ParagraphToolbar is removed.
