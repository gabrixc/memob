Canvas Interaction Summary

  • Snap-to-grid — 8px grid (configurable), yellow alignment guides appear when edges/centers align with other objects
  • Anti-ghosting drag — Fabric.js renders one solid object while dragging; pointer offset from object origin is preserved so it doesn't "jump" to cursor center
  • Multi-select — rubber-band drag-select or Shift+click; selected group moves/resizes together
  • Field drop — dragging from right pane creates a text object with {{field_name}} content, immediately editable and snappable
  • Keyboard shortcuts — Ctrl+C/X/V for copy/cut/paste, Delete to remove, Ctrl+Z/Y for undo/redo, arrow keys for nudge
