Layout Summary

  • Top bar — app menu, Save Template, Preview with Data, Export buttons
  • Left toolbar (56px) — Select, Text Box, Line, Table, Image, Rectangle, Page Break, Copy/Cut/Paste, Grid toggle
  • Canvas (center) — Fabric.js canvas with A4 page, dot-grid background, snap indicator, properties bar showing X/Y/W/H of selected object
  • Right pane (200px) — DB field browser grouped by table, drag onto canvas to insert placeholder; active record selector at bottom
  • Status bar — page count, zoom, canvas size, object count, grid info
